Text Formatting Elements in HTML

  1. <b> Bold Text
    for example, MCA23

  2. <strong> Important Text
    for example, Chandrayaan-3 is an important achievement for India.

  3. <i> Italic Text
    for example, Amigos is friend in spanish.

  4. <u> Underline Text
    for example, won is now in right to left direction.

  5. <em> Emphasized Text
    for example, GitHub is the best tool for remote opportunities.

  6. <mark> Marked Text
    for example, Communication is the key for expected negotiation.

  7. <small> Small Text
    for example, Wars must end, and Peace must prosper.

  8. <del> Deleted Text
    for example, DON'T QUIT!

  9. <ins> Inserted Text
    for example, Push 3 and Pop, you get 3.

  10. <sub> SubscriptText
    for example, H2O is important for us.

  11. <sup> SuperscriptText
    for example, 22 is 4.

  12. <bdo> Bidirectional Over Ride Text
    ffor example, now is originally now!
