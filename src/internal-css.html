Introduction to Internal CSS

Internal CSS is used to define a style for a single HTML page.

An internal style sheet may be used if one single page has a unique style.

To use internal CSS, you must use the <style> element in the <head> section.

Internal CSS is defined within the <style> element, inside the <head> section of an HTML page:

        <head>
            <style>
                body {
                    background-color: linen;
                }
                h1 {
                    color: maroon;
                    margin-left: 40px;
                }
            </style>
        </head>
